进行转账操作

要想进行转账操作，前面提到的POSTING key就不行了，这里需要获取另外一个key，也就是”Active key”，同样，点”Wallet” -> ”Permissions”，就会看到Active key，不是系统默认显示的这个，点“Login to see private key”，登陆后就可以看到一个以数字5开头的密码了。

有了这个key，就可以进行转账操作了。下面首先来调戏一下booster这个机器人，它和randowhale类似，你转钱给它，它就会Upvote你。下面用前面发过的测试贴来验证一下booster的upvote是不是够给力。
